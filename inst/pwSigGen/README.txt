pwSigGen -- folder of material related to creating pathway signatures based on Bild

srcsig.txt -- derived from supplementary data of Bild Nature Jan 2006 
  they give a pdf file with probe set ids for different signatures, this
  file gives the lines devoted to Src

E2F3sig.txt -- derived from supplementary data of Bild Nature Jan 2006 ;
  the lines devoted to E2F3

Other files in this folder deal with these text files to generate
signature vectors.
